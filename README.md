## 🔧 Installation

```bash
git clone https://github.com/adarsh-singh17/Portfolio.git
cd Portfolio

python -m venv venv
venv\Scripts\activate   # Windows

pip install -r requirements.txt
python app.py
```

👉 Open: http://127.0.0.1:5000
